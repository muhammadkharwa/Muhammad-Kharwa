# Calc v2
Calc v2 is a simple command-line calculator written in Python. It lets the user enter two numbers, choose an operator, see the result, and then decide whether to run another calculation.

# Features
* Supports addition, subtraction, multiplication, and division
* Accepts decimal numbers
* Checks that number inputs are valid
* Checks that the operator is one of +, -, *, or /
* Prevents division by zero
* Lets the user perform multiple calculations in one session

# Notes
* If the user enters something that is not a number, the program asks for a valid number again.
* If the user enters an unsupported operator, the program asks for a valid operator again.
* If the user tries to divide by zero, the program displays "Cannot divide by zero" and asks whether to continue.
